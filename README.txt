Java Mapreduce program that allows you to perform XML Parsing in Hadoop HDFS
Load the file in Eclipse
Give the tag name from which you need to extract information
Generate JAR File with custom properties file or use the properties file added
Load the JAR into local system. Make sure that the properties file is in the local system as well
Load the XML file in hdfs 
Execute the JAR File in onto the XML file. 
View the output in the path you specified.
